## 📌 Overview
This project analyzes sales data using SQL and Power BI, covering the complete data workflow from storage to visualization.

The dashboard provides insights into sales performance, revenue trends, customer behavior, and product performance to support data-driven decision-making.

## 🔄 Workflow

### 🗄️ Data Creation (SQL)
- Created structured tables for sales, customers, and products  
- Stored and managed data efficiently  

### 🔗 Data Integration
- Connected SQL database to Power BI  
- Imported datasets for analysis  

### 🧹 Data Cleaning & Transformation
- Handled missing values  
- Standardized data formats  
- Created calculated columns and measures  

### 🧠 Data Modeling
- Built relationships between tables  
- Designed a **star schema** for performance  

### 📊 Dashboard Development
- Developed interactive visuals  
- Added filters, slicers, and drill-down features
